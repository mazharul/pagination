This is a jQuery plugin for pagination with example php and html script. PHP and html scripts are based on smarty and I am so lazy to make a normal script :P. Feel free to modify and use on any of your project. This example scripts assume you know the basic of smarty templating engine. You can use the script in your page without smarty as well.

Usage

You need to load both jquery and the plugin in your page and then initialize the pagination plugin script. Initialized below js script is based on smarty templating engine.

$(document).ready(function(){ldelim}
  $("#paginationItems").paginate({ldelim}
    urlToFetchdata: "/paging.php?act=pagination",
    totalPage:{$pages} //Total pages assigned by smarty from php script
  {rdelim});
{rdelim});

Example of a html page in smarty:

<!-- ajax result holder div -->
<div id="paginationItems">
{*ajax results holder*}
</div>

<!--pagination-->
    {if $pages >1}
    <div class="pagination-container" style="margin: 0 0 0 auto; text-align: right;">
    <div id="prev" class="disabled">&lt; Prev</div>
    <ul id="pagination">

    {section name=sec start=1 loop=$pages+1 step=1}
         <li id='page_{$smarty.section.sec.index}' class="paginationLiClass">{$smarty.section.sec.index}
        <input class="currentPage" type="hidden" value="{$smarty.section.sec.index}" /></li>
       <li class="paginationLiDivider">{if $smarty.section.sec.index< $pages}|{else}&nbsp;&nbsp;{/if}</li>
    {/section}
    </ul>
    <div id="next" style="float:left;" class="paginationDiv">Next &gt;</div>
    <div class="clear"></div>
    </div>
    {/if}
